Assembly Code
Keyboard Input PORT 0
Display Output PORT 16

Assembly Instruction Reference

Quick Info

Registers: A (accumulator), X, Y (index), SP (auto)
Immediate range: −256 … +255 (9-bit signed)
Memory: 0x000–0x3FF (1024 × 16-bit words)

Example — Echo keyboard to display

loop: IN 1 ; A = KBD_STATUS (bit 0 = data ready) ANDI 1 ; isolate data-ready bit BRZ loop ; no char yet → keep polling IN 0 ; A = KBD_DATA (ASCII), clears latch OUT 16 ; write A to display BRA loop ; next character END

Control Flow

ENDHalt execution
JMP addrUnconditional jump to address
RETReturn: pop PC from stack
BRA addrBranch always (relative to label)
BRZ addrBranch if A = 0 (Zero flag)
BRN addrBranch if A < 0 (Negative flag)
BRC addrBranch if Carry flag set
BRO addrBranch if Overflow flag set
BGT addrBranch if A > operand (signed)
BLT addrBranch if A < operand (signed)
BGE addrBranch if A ≥ operand (signed)
BLE addrBranch if A ≤ operand (signed)
BEQ addrBranch if equal (A = 0 after CMP)
BNE addrBranch if not equal

Memory Access

LD X, addrX = mem[addr]
LD Y, addrY = mem[addr]
ST X, addrmem[addr] = X
ST Y, addrmem[addr] = Y
PUSH XPush X onto stack; SP--
PUSH YPush Y onto stack; SP--
POP XPop top of stack into X; SP++
POP YPop top of stack into Y; SP++

Move / Load Immediate

MOVI immA = sign-extended 9-bit immediate
MOV X, immX = immediate value
MOV Y, immY = immediate value
MOVR A, XA = X
MOVR A, YA = Y
MOVR X, AX = A
MOVR Y, AY = A
MOVR X, YX = Y
MOVR Y, XY = X

Arithmetic

ADD X / ADD YA = A + X (or Y)
SUB X / SUB YA = A − X (or Y)
MUL X / MUL YA = A × X (or Y)
DIV X / DIV YA = A ÷ X (or Y)
MOD X / MOD YA = A mod X (or Y)
ADDI immA = A + immediate
SUBI immA = A − immediate
MULI immA = A × immediate
DIVI immA = A ÷ immediate
MODI immA = A mod immediate
INC X / INC YX++ or Y++ (does not affect A)
DEC X / DEC YX-- or Y-- (does not affect A)

Shift & Rotate

LSL X / LSL YA <<= X (logical shift left)
LSR X / LSR YA >>= X (logical shift right)
RSL X / RSL YA <<= X (rotate left)
RSR X / RSR YA >>= X (rotate right)
LSLI immA <<= imm (logical shift left)
LSRI immA >>= imm (logical shift right)
RSLI immA <<= imm (rotate left)
RSRI immA >>= imm (rotate right)

Logical

AND X / AND YA = A & X (or Y)
OR X / OR YA = A | X (or Y)
XOR X / XOR YA = A ^ X (or Y)
NOT X / NOT YA = ~X (or ~Y)
ANDI immA = A & immediate
ORI immA = A | immediate
XORI immA = A ^ immediate
NOTI immA = ~immediate

Compare & Test

CMP X / CMP YSet flags from A − X (discard result)
TST X / TST YSet flags from A & X (discard result)
CMPI immSet flags from A − immediate
TSTI immSet flags from A & immediate

I/O

IN portA = I/O port value (10-bit port address)
OUT portI/O port = A (10-bit port address)

Interrupts

EIEnable interrupts (set I flag)
DIDisable interrupts (clear I flag)
IRETReturn from interrupt: pop PC & FLAGS, re-enable
WAITHalt CPU until an interrupt fires

Accelerator

MINEStart SHA-256 mining (X=data, Y=nonce, A=target); result in ports 64–65
